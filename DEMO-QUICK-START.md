# 🚀 빠른 데모 가이드

## 1️⃣ 웹 페이지 접속

```
http://136.110.134.8/
```

## 2️⃣ 로그인

- 회원가입 후 로그인
- 또는 테스트 계정 사용

## 3️⃣ 오른쪽 상단 모니터링 위젯 확인

- CPU 사용률
- 메모리 사용률
- **🖥️ 활성 인스턴스: 2개** ← 현재 상태

## 4️⃣ 부하 테스트 시작

모니터링 위젯 하단:

1. **🚀 부하 시작** 버튼 클릭
2. 초당 100개 요청 전송 시작
3. 통계 확인 (요청 수, 성공, 실패)

## 5️⃣ 오토스케일링 확인 (5-10분 대기)

- **활성 인스턴스**: 2개 → 3개 → 4개 → 5개
- GCP Console에서도 확인 가능

## 6️⃣ 부하 중지

- **⏹️ 부하 중지** 버튼 클릭
- 10-15분 후 인스턴스 2개로 감소

---

## 📊 GCP Console 확인

**인스턴스 그룹**:

- Compute Engine → 인스턴스 그룹 → football-app-group-v2
- 모니터링 탭: CPU 사용률, 인스턴스 개수
- 멤버 탭: 실시간 인스턴스 목록

**로드밸런서**:

- 네트워크 서비스 → 부하 분산 → football-load-balancer
- 모니터링 탭: 요청 수, 트래픽 분산

---

## ⚠️ 주의사항

- 인스턴스 개수는 1분마다 업데이트
- 오토스케일링은 5-10분 소요
- 부하 테스트는 10분 이상 실행 권장
